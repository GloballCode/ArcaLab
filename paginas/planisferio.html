ArcaLab

Planisfério Estelar Interativo

Este é um projeto desenvolvido por Gabrielly Barros, que permite explorar as constelações e estrelas visíveis no céu noturno. O projeto foi criado utilizando a plataforma Scratch.

Tutorial - Como jogar

1. Após clicar na bandeirinha verde, clique na bandeira dos EUA para escolher o idioma, clicando até encontar a bandeira do país falante do idioma desejado. Em seguida, clique no botão 'Start'

2. Escolha entre o Hemisfério Norte e o Hemisfério Sul

3. Clice na constelação para ver o nome dela.

Clique em 'Sair' para sair do hemisfério.

4. Ao sair, voltará à página 'Escolha o hemisfério'. Escolha o hemisfério desejado novamente.

5. Divirta-se!
